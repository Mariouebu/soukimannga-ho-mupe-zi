管理者本番ログイン

⚠️ FirebaseのAuthenticationで設定した本物の管理者用メールアドレス・パスワードを入力してください。

Firebaseデータベースの有効化が必要です

プログラムは正常に連動していますが、お使いのFirebaseプロジェクトで「Cloud Firestore データベース」のセットアップが完了していません。

解消のための3ステップ設定手順

  1. Firebase Console を開きます。
  2. 作成したプロジェクト（my-shop-com）を選択します。
  3. 左メニューの 「構築（Build）」 ➔ 「Firestore Database」 をクリックします。
  4. 画面中央の 「データベースを作成」 ボタンを押します。
  5. 位置を「asia-northeast1（東京）」にし、セキュリティルールを一時的に「テストモードで開始」に設定して作成（有効化）します。

エラー詳細: [code=permission-denied] Firestore API has not been enabled or used yet.
